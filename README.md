# Account service

Serviço responsável por gerenciar a conta do usuário.

## Tecnologias

[nestjs](https://nestjs.com/)

[Banco de dados MongoDB](https://www.mongodb.com/try/download/community?tck=docs_server)

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch/dev mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Executando testes

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
